# API — Обзор

## Базовый URL

```
https://api.CareerMap.com/api/v1
```

## Аутентификация

> **Шаблон раздела.** Описание схемы аутентификации добавляется при реализации.

Предполагается использование Bearer-токена в заголовке `Authorization`:

```http
Authorization: Bearer <token>
```

## Разделы API

| Тег | Описание |
|---|---|
| Специализации | Получение списка и информации о специализациях |
| Материалы | Получение уроков и их деталей |
| Прогресс | Оценка материалов и отметка о прохождении |

## Полная спецификация (Redoc)

Интерактивная документация API сгенерирована из OpenAPI-спецификации и доступна по ссылке в навигации: **API Reference**.

## Коды ответов

| Код | Значение |
|---|---|
| 200 | Успешный ответ |
| 404 | Ресурс не найден |
| 500 | Внутренняя ошибка сервера |

## Управление домашними заданиями

> **Шаблон раздела.** Эндпоинты для загрузки и проверки ДЗ будут добавлены в следующей версии API.

Планируемые эндпоинты:

- `POST /homework` — загрузить домашнее задание
- `GET /homework?status=pending` — список заданий на проверку
- `POST /homework/{id}/grade` — выставить оценку
