# Карьерный навигатор с экспертным контентом

## Описание продукта

**Карьерный навигатор** — образовательная платформа, которая помогает пользователям выстраивать карьерную траекторию в digital-профессиях через структурированные учебные материалы, проверку домашних заданий экспертами и интерактивную карьерную карту.

## Основные возможности

| Функция | Описание |
|---|---|
| Карьерная карта | Выбор специализации и уровня, прохождение материалов |
| Домашние задания | Отправка и проверка ДЗ экспертами с оценкой |
| Проверка грейда | Тест для определения текущего уровня знаний |
| Обсуждения | Форум по темам с участием экспертов |
| Размещение вакансий | Витрина вакансий от работодателей |

## Роли пользователей

- **Пользователь (User)** — проходит обучение, отправляет ДЗ, участвует в обсуждениях
- **Эксперт (Expert)** — проверяет ДЗ, отвечает на вопросы, участвует в обсуждениях
- **Автор (Author)** — создаёт и публикует учебные материалы на платформе

## Технологический стек

- **Frontend:** Web-приложение (SPA)
- **Backend:** REST API
- **База данных:** PostgreSQL
- **Хранилище:** S3-совместимое объектное хранилище
- **Почтовый сервис:** SMTP / внешний провайдер
